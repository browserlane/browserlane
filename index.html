browserlane

Browser automation for humans and AI agents — one Rust binary, over WebDriver BiDi.

Install — macOS / Linux

curl -fsSL https://browserlane.com/install.sh | sh

Install — Windows (PowerShell)

irm https://browserlane.com/install.ps1 | iex

Then

bl go https://example.com        # drive a browser from the CLI
bl add-mcp claude                # wire the MCP server into your agent
