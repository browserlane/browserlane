Pause execution for a number of milliseconds

**Usage**

```bash
bl sleep [ms] [flags]
```

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl sleep 1000
# Wait 1 second

bl sleep 500
# Wait 500ms
```
