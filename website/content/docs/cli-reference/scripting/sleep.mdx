Pause execution for a number of milliseconds

**Usage**

```bash
bl sleep <ms>
```

**Arguments**

| Argument | Description |
| --- | --- |
| `&lt;ms&gt;` | Number of milliseconds to pause |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash
bl sleep 1000
# Wait 1 second

bl sleep 500
# Wait 500ms
```
