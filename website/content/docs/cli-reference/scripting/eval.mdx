Evaluate a JavaScript expression (optionally navigate to URL first)

## Usage

```bash
bl eval [args...] [OPTIONS]
```

## Arguments

| Argument | Description |
| --- | --- |
| `[args...]` | [url] expression — JS to evaluate, optionally preceded by a URL to navigate first |

## Options

| Option | Description |
| --- | --- |
| `--stdin` | Read expression from stdin |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |

## Global options

| Option | Description |
| --- | --- |
| `-v, --verbose` | Enable debug logging |

## Examples

```bash
bl eval "document.title"
# Evaluates on current page

bl eval https://example.com "document.title"
# Navigates to URL first, then evaluates

bl eval --stdin
# Read expression from stdin, e.g. echo 'document.title' | ... (avoids shell quoting issues)
```
