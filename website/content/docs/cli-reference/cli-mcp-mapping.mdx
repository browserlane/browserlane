browserlane is one engine behind two surfaces: the human `bl` CLI and the agent-facing MCP server. They are **not 1:1**. Some CLI verbs are renamed on the MCP side (`bl go` → `browser_navigate`); some single commands split into get/set tool pairs (`bl viewport` → `browser_get_viewport` + `browser_set_viewport`); the page-clock controls are MCP-only; and the install / daemon / agent-wiring plumbing is CLI-only. This page is generated from `bl v0.1.3`.
