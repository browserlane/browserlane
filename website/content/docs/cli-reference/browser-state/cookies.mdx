Manage browser cookies

**Usage**

```bash
bl cookies [args...]
bl cookies <command>
```

**Arguments**

| Argument | Description |
| --- | --- |
| `[args...]` | Cookie name and value to set (omit both to list all cookies) |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash
bl cookies
# List all cookies

bl cookies "session" "abc123"
# Set a cookie with name and value
```
