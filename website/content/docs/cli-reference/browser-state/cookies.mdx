Manage browser cookies

**Usage**

```bash
bl cookies [name] [value] [flags]
bl cookies [command]
```

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

