Get or set the browser viewport size

**Usage**

```bash
bl viewport [width] [height] [flags]
```

**Flags**

| Flag | Description |
| --- | --- |
| `--dpr float` | Device pixel ratio (e.g., 2 for Retina) |

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl viewport
# {"width":1280,"height":720,"devicePixelRatio":1}

bl viewport 1280 720
# Set viewport to 1280x720

bl viewport 375 812 --dpr 3
# Simulate iPhone X viewport
```
