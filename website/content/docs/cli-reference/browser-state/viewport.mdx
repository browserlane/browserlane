Get or set the browser viewport size

**Usage**

```bash
bl viewport [args...] [OPTIONS]
```

**Arguments**

| Argument | Description |
| --- | --- |
| `[args...]` | Width and height to set (omit both to print the current viewport) |

**Options**

| Option | Description |
| --- | --- |
| `--dpr <value>` | Device pixel ratio (e.g., 2 for Retina) |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash
bl viewport
# {"width":1280,"height":720,"devicePixelRatio":1}

bl viewport 1280 720
# Set viewport to 1280x720

bl viewport 375 812 --dpr 3
# Simulate iPhone X viewport
```
