Export or restore browser state (cookies, localStorage, sessionStorage)

**Usage**

```bash
bl storage [flags]
bl storage [command]
```

**Flags**

| Flag | Description |
| --- | --- |
| `-o, --output string` | Output file path |

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl storage
# Print state as JSON

bl storage -o state.json
# Save state to file
```

### `bl storage restore`

Restore browser state from a JSON file

**Usage**

```bash
bl storage restore [path] [flags]
```

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl storage restore state.json
# Restore cookies and storage from saved state
```
