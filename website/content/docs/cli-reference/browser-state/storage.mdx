Export or restore browser state (cookies, localStorage, sessionStorage)

**Usage**

```bash
bl storage [OPTIONS]
bl storage <command>
```

**Options**

| Option | Description |
| --- | --- |
| `-o, --output <value>` | Output file path |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash
bl storage
# Print state as JSON

bl storage -o state.json
# Save state to file
```

### `bl storage restore`

Restore browser state from a JSON file

**Usage**

```bash
bl storage restore <path>
```

**Arguments**

| Argument | Description |
| --- | --- |
| `&lt;path&gt;` | Path to a JSON state file to restore |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash
bl storage restore state.json
# Restore cookies and storage from saved state
```
