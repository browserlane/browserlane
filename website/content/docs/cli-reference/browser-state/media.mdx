Override CSS media features

## Usage

```bash
bl media [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `--color-scheme <value>` | Color scheme: light, dark, no-preference |
| `--reduced-motion <value>` | Reduced motion: reduce, no-preference |
| `--forced-colors <value>` | Forced colors: active, none |
| `--contrast <value>` | Contrast: more, less, no-preference |
| `--media <value>` | Media type: screen, print |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |

## Global options

| Option | Description |
| --- | --- |
| `-v, --verbose` | Enable debug logging |

## Examples

```bash
bl media --color-scheme dark
# Enable dark mode

bl media --reduced-motion reduce
# Reduce motion

bl media --color-scheme light --forced-colors active
# Override multiple features
```
