### `bl page close`

Close a browser page by index (default: current page)

**Usage**

```bash
bl page close [index] [flags]
```

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl page close
# Close current page (index 0)

bl page close 1
# Close page at index 1
```

### `bl page new`

**Usage**

**Global flags**

| Flag | Description |
| --- | --- |
