**Usage**

```bash

```

**Arguments**

| Argument | Description |
| --- | --- |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash

```
