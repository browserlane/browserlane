Press a key or key combination

**Usage**

```bash
bl keys [keys] [flags]
```

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl keys Enter
# Press Enter

bl keys "Control+a"
# Select all

bl keys "Shift+Tab"
# Shift+Tab to previous field
```
