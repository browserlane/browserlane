Select an option in a &lt;select&gt; element

**Usage**

```bash
bl select <selector> <value>
```

**Arguments**

| Argument | Description |
| --- | --- |
| `&lt;selector&gt;` | CSS selector or @ref of the &lt;select&gt; element |
| `&lt;value&gt;` | Value of the option to select |

**Global options**

| Option | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `-v, --verbose` | Enable debug logging |
| `--json` | Output as JSON |

**Examples**

```bash
bl select "select#color" "blue"
# Select "blue" in the color dropdown
```
