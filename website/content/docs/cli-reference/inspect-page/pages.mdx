List all open browser pages

**Usage**

```bash
bl pages [flags]
```

**Global flags**

| Flag | Description |
| --- | --- |
| `--headless` | Hide browser window (visible by default) |
| `--json` | Output as JSON |
| `-v, --verbose` | Enable debug logging |

**Examples**

```bash
bl pages
# [0] https://example.com
# [1] https://google.com
```
